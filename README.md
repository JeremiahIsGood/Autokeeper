# 计车器

一个用于快速记录汽车品牌出现次数的离线网页 App。

## 当前功能

- 广东省、香港、澳门区县级地区选择
- 地区搜索和全部地区统计
- 排行榜独立页面，记录页面保留快速记录操作
- 排行榜支持区县、城市、省级、全部地区范围
- 排行榜支持今日、本周、历史月份、历史年份、全部时间
- 常用、中国、日本、美国、德国、英国、韩国、法国、意大利、其他品牌分组
- 常用分组自动显示近 7 天最常点击的 9 个品牌
- 搜索品牌后自动定位到所在分组
- 长按品牌可进入本组排序模式，常用分组除外
- 长按品牌可查看记录或减一
- 点击品牌自动记录品牌、地区和时间
- 今日、本周、本月、本年、全部排行榜
- 撤销上一次记录
- 本地保存
- JSON 备份导出
- JSON 备份导入

## 打开方式

电脑上可以直接打开 `index.html` 预览。

如果要在 iPhone 上添加到主屏幕，需要通过本地网址或部署网址访问，而不是直接打开文件。之后可以用 Safari 打开网页，点分享按钮，再选择“添加到主屏幕”。
